# Hospicare Agency

Professional informational website for a wholesale medical agency based in Nashik, Maharashtra, India.

## Project status

Frontend pages, shared components, and backend email endpoint are implemented.

## Tech stack

- React 18 + Vite (frontend)
- Node.js + Express (email sending only)
- Firebase Firestore, Auth, Storage
- Nodemailer (Gmail SMTP)

## Structure

- `client/` — React frontend
- `server/` — Express backend (contact email)

## Environment variables

Fill in `.env` files in `client/` and `server/` with your actual credentials.
